MedSupply: Hospital Tests Overview
Welcome! The Tests Inventory page displays all the medicines that hospitals have sent for testing. You can view the complete list of medicines in the inventory, and clicking on any item will reveal additional information and reverse engineering details of the selected medicines.
MedSupply:Key Features Of Testing Page of Hospitals:
  • Request New Test
  • Test Status
  • Tests Details
MedSupply: Request New Test:
This section allows hospitals to add new test requests for a specific medicine. Users can submit test requests to available laboratories, using an interface designed to streamline the process step-by-step.
MedSupply: Test Status & Test Details:
This section displays the Test Status and Details for all requested tests. Users can view the current status of each test, marked with tags such as Pending, Completed, or Rejected. By clicking on More Details, users can access a weekly test trends graph and a comprehensive report that includes information such as the requested tests, test results, and any additional details related to the test process. A Download Report button will also be available, allowing users to download detailed reports of their test requests for further analysis. Additional details such as the laboratory information (including the lab's address, ratings, and contact details) are provided. Instead of shipping information, users can view the tests requested for a specific medicine, along with a status timeline showing each stage of the testing process. This timeline is expandable, providing a detailed overview of the test progression.
MedSupply:Future Updates:
We are planning to enhance the dashboard by improving responsiveness across mobile and all devices, implementing a more advanced user interface with enhanced functionalities. Additionally, we will integrate a sample database for testing and demonstration purposes. A new View Laboratories page will also be added, displaying all available labs that offer testing services, providing comprehensive information for easy access and selection.
